# **Tareeq - طريق**  
  

<div align="right">طريق هو مشروع تعاوني مفتوح المصدر يهدف الى تسهيل الطريق نحو تعلم المهارات التقنية والبرمجة عن طريق امداد المستخدمين بخرائط توجيهية ودروس ومصادر تعليمية باللغة العربية</div>  
  

Tareeq is an open-source project to make the road easy for learners by providing roadmaps, guides & resources for them in Arabic language.
